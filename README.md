# The Star Jar - 愿星罐 🌟

一个基于物理引擎的情绪手账 Web 应用，将你的心情折叠成星星，收藏在玻璃罐中。

## ✨ 特性

- 🎨 **治愈系设计**：温暖的纸质风格，柔和的色彩搭配
- ⭐ **折星星交互**：将文字心情折叠成彩色星星
- 🫙 **物理引擎**：使用 Matter.js 实现真实的物理碰撞和重力效果
- 💾 **本地存储**：所有数据保存在浏览器本地，保护隐私
- 📱 **响应式设计**：支持桌面和移动设备

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

## 🎮 使用方法

1. 点击"写下心情"按钮
2. 在纸条上输入你的心情文字
3. 点击"折叠"按钮，观看星星折叠动画
4. 星星会自动掉入玻璃罐中
5. 在罐子视图中，可以拖拽星星、查看回忆
6. 点击星星可以展开查看完整内容

## 🛠️ 技术栈

- HTML5 / CSS3
- JavaScript (ES6+)
- Matter.js (物理引擎)
